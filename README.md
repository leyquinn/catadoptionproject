# Cat Adoption Project

A modern web application designed to help users explore and adopt cats. Built with React and Vite for fast development and optimized performance.

## Features

- **Browse Cats**: Explore available cats for adoption with detailed information
- **Adoption Process**: Learn about our adoption process and requirements
- **About Us**: Discover the mission and values of our organization
- **User-Friendly Interface**: Clean, responsive design for an excellent user experience

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/leyquinn/catadoptionproject.git
cd catadoptionproject
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173` (or the URL shown in terminal)

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
